## SkillCraft Technology Internship - Task 2

### Guess the Number Game

A simple web application where the computer generates a random number between 1 and 100, and the user tries to guess it.

## Features

- Random number generation
- Too High / Too Low hints
- Attempts counter
- Play Again button
- Responsive interface

## Technologies Used

- HTML
- CSS
- JavaScript
